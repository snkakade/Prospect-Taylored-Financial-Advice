# Taylored Financial Advice Website

Static responsive redesign for Taylored Financial Advice using the supplied concept direction, logo and founder image.

## Run locally

Open `index.html` directly in a browser. If Python is installed, you can also serve the folder:

```powershell
python -m http.server 5173
```

Then visit `http://localhost:5173`.

## Launch notes

- Contact form is a placeholder and does not send enquiries.
- Owner-approved imagery, compliance wording, privacy policy, cookie banner and complaints procedure still need final approval before launch.
- The page includes responsive layout, sticky glass navigation, mobile menu, scroll reveal, magnetic CTAs, tilt cards, animated metrics, FAQ accordion and clarity-score planner.
- Smooth scrolling uses Lenis from a CDN when the visitor is online. If it cannot load, the page falls back to native browser scrolling without breaking interactions.
